# RetailPulse – AI-Powered Customer Analytics & Demand Forecasting

## Overview

RetailPulse is an end-to-end Data Science and Analytics platform developed for retail businesses.

The platform provides:
- Demand Forecasting
- Customer Segmentation
- Churn Prediction
- Inventory Optimization
- Interactive Dashboard
- MLflow Tracking
- Docker & Kubernetes Deployment

## Features

### Demand Forecasting
- Prophet Model
- LSTM Model
- Hybrid Forecasting

### Customer Analytics
- RFM Analysis
- Customer Segmentation

### Churn Prediction
- XGBoost Model
- SHAP Explainability

### Inventory Optimization
- Reorder Quantity Recommendations

### Dashboard
- Forecast Dashboard
- Customer Analytics Dashboard
- Inventory Dashboard
- Alerts Dashboard

## Technology Stack

- Python
