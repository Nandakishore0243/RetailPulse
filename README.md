# 📊 RetailPulse – AI-Powered Customer Analytics & Demand Forecasting

## Overview

RetailPulse is an end-to-end Data Science and Analytics platform designed to help retail businesses make data-driven decisions through customer analytics, demand forecasting, churn prediction, and inventory optimization.

The platform combines machine learning models, business intelligence techniques, and interactive dashboards to provide actionable insights for retail operations.

---

## Key Features

### 📈 Demand Forecasting

* Prophet Forecasting Model
* LSTM Forecasting Model
* Hybrid Forecasting Approach

### 👥 Customer Analytics

* RFM Analysis
* Customer Segmentation using K-Means Clustering
* Customer Behavior Insights

### 🔄 Churn Prediction

* XGBoost Classification Model
* SHAP Explainability Analysis

### 📦 Inventory Optimization

* Reorder Point Calculation
* Safety Stock Estimation
* Inventory Recommendations

### 📊 Interactive Dashboard

* Forecasting Dashboard
* Customer Analytics Dashboard
* Inventory Dashboard
* Alerts & Monitoring Dashboard
